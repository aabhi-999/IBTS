# IBTS Frontend Guide

## Technology Stack
- React 19.1.1
- React Router DOM 7.8.2
- Tailwind CSS
- Custom Animations

## Features
- ✅ Splash Screen Animation
- ✅ Bus Search & Listing
- ✅ Seat Booking System
- ✅ Responsive Design
- ✅ Error Handling
- ✅ Loading States

## Components
- `SplashScreen` - Loading animation
- `BusList` - Display available buses
- `BookingForm` - Seat booking interface
- `SeatSelection` - Interactive seat map
- `ErrorBoundary` - Error handling

## Development
```bash
cd frontend
npm start
```

## Build
```bash
npm run build
```
